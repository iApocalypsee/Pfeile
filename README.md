Pfeile
======

A little "fantasy" game.

Used libraries
==============
  - json4s (https://github.com/json4s/json4s)
  - if necessary install SCALA (http://www.scala-lang.org/) or update JAVA to Version 8 (https://www.java.com)

Launch
======

For launching the program, do

```
cd bin
scala general.Main
```
